Um Rules
Place your castle on your half. Click an empty square twice to place a node, or click a friendly node to start tracing a node-to-node path chain. Each segment must be orthogonal and may use only one corner. Paths only cross enemy paths perpendicularly, and only if the crossing path is shorter, which cuts them. Nodes next to each other are not connected unless a path is actually drawn between them. Enemy nodes die if snuffed on all orthogonal sides or trapped inside a connected enclosure. Infinite Board expands when edge nodes are placed. Win by the selected kill target, or by killing the enemy castle in To the Death.
